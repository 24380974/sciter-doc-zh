Integer(整数)值

所有的整数值都是这个类的"对象"。

常量

MIN
- integer, 整数类型的最小值。
MAX
- integer, 整数类型的最大值。

方法

toFloat

( ) : float

架构整个转换成浮点数。

toInteger

( ) : int

简单的返回整数值。

toString
( [radix: int] ) : string

返回整数的字符串型，radix参数指定进制，取值10(默认)、8、16。

toHtmlString

( [radix: int] ) : string

返回转义了html规则的字符串。是toString()方法的别名。

toUrlString

( [radix: int] ) : string

返回转义了url规则的字符串。是toString()方法的别名。

valueOf
() : int

简单的返回整数值本身。

min

( ... ) : int

静态方法 - 返回参数中的最小值。如Integer.min(1, 2, 3)将会返回1。

max

( ... ) : int

静态方法 - 返回参数中的最大值。如Integer.min(1, 2, 3)将会返回3。

limit

( min:int, max:int ) : int

该函数的实现代码如下:

function Integer.limit(min, max) 
{
   if( this < min ) return min;
   if( this > max ) return max;
   return this;
}
